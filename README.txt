الملفات في هذا المجلد
──────────────────────
  build.py         → سكريبت يُعدّل الـ HTML أوتوماتيكياً (خطوة 1)
  content.json     → كل البيانات القابلة للتعديل (أرقام، أخبار، روابط...)
  cms-loader.js    → يقرأ content.json ويحدّث الصفحة عند الفتح
  admin/
    index.html     → لوحة Decap CMS
    config.yml     → إعدادات اللوحة (عدّل repo فيه)

 الخطوات (مرة واحدة فقط)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① ضع ملف الموقع الأصلي (wataawano.org.html) داخل هذا المجلد.

② في Terminal، شغّل:
      python build.py wataawano.org.html
   سينتج ملف index.html جاهز.

③ في admin/config.yml عدّل السطر:
      repo: YOUR_GITHUB_USERNAME/wataawano
   بمستخدمك ومستودعك الفعليين على GitHub.

④ ارفع كامل هذا المجلد إلى مستودع GitHub.

⑤ اربط المستودع بـ Netlify (New site from Git).
   • في Netlify Dashboard → Site Settings → Identity → Enable.
   • ثم Registration → Invite Only.
   • أرسل دعوة لبريدك الإلكتروني.

⑥ افتح yourdomain.com/admin/ وسجّل دخولك.

• افتح /admin/ → ستجد قسماً واحداً "محتوى الموقع".
• عدّل ما تريد (أرقام الأثر، الأخبار، IBAN، روابط التواصل...).
• اضغط Publish → يُحدَّث content.json في GitHub تلقائياً.
• Netlify يُعيد نشر الموقع خلال ثوانٍ.
